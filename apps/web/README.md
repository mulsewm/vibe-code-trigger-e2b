# Vibe Code Web App

Frontend Next.js application for the Vibe Code platform.

## Features

- Code editor with syntax highlighting
- Real-time output streaming
- Multiple language support (Python, JavaScript, TypeScript, Bash)
- Clean, modern UI

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

- `NEXT_PUBLIC_API_URL` - Backend API URL (default: http://localhost:3001)

## Build

```bash
npm run build
npm start
```
